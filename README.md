# Martex - Premium SaaS & App Landing Page Template

Martex is a state-of-the-art landing page template built with **React**, **Vite**, and **Tailwind CSS**. It features a modern, clean design with advanced theming capabilities and interactive UI components.

---

## 🚀 Getting Started

Follow these simple steps to get your development environment up and running.

### 1. **Prerequisites**
Ensure you have [Node.js](https://nodejs.org/) (v16.x or later) and **npm** installed on your machine.

### 2. **Install Dependencies**
In your project directory, run the following command to install all necessary packages:
```bash
npm install
```

### 3. **Start the Development Server**
Launch the local development server with Vite:
// turbo
```bash
npm run dev
```

### 4. **Open in Browser**
Once the server starts, you'll see a local URL in your terminal. By default, you can open your browser and navigate to:
👉 **[http://localhost:5173](http://localhost:5173)**

## ✨ Key Features

- **🎨 Multi-Theme Color Switcher**: Change the primary color globally with a single click in the sidebar drawer.
- **✨ Interactive Custom Cursor**: A smooth, springy mouse follower that reacts to links and buttons.
- **🌗 Dark Mode Support**: Fully optimized for both light and dark environments.
- **📱 Responsive Design**: Perfectly tailored for Desktop, Tablet, and Mobile devices.
- **🚀 Ultra Fast Performance**: Powered by Vite and Tailwind CSS for instant loading and hot module replacement (HMR).
- **🎭 Framer Motion Animations**: Premium, smooth transitions and reveal-on-scroll effects.

---

## 🛠 Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

---

## 📁 Project Structure

- `src/components/`: Reusable UI elements (StyleSwitcher, Cursor, Header, etc.).
- `src/pages/`: Main page views (Landing, About, Careers, Contacts, etc.).
- `src/index.css`: Global styles and dynamic CSS variables for theming.
- `tailwind.config.js`: Tailwind configuration including custom theme colors.
